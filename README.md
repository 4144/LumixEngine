#LumixEngine
=========

[Latest executables](https://ci.appveyor.com/project/nem0/lumixengine/build/artifacts)

3D Game Engine

Features:
  * multiplatform (PC, consoles, phones, tablets, ...)
  * editor included
  * easy to use, fast and flexible 
  * open source
  * component based
  * data driven
